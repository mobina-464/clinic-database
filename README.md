# 🦷 سیستم مدیریت کلینیک دندان‌پزشکی (Dental Clinic Management System)

این پروژه شامل طراحی و پیاده‌سازی پایگاه داده یک کلینیک دندان‌پزشکی است که تمامی فرآیندهای مربوط به بیماران، نوبت‌دهی، پزشکان، خدمات درمانی و پرداخت‌های مالی را پوشش می‌دهد.

## 📋 ویژگی‌های پروژه
- طراحی کامل بر اساس نمودار **ER** (موجود در مستندات).
- پیاده‌سازی روابط **One-to-Many** و **Many-to-Many**.
- استفاده از ویژگی **Auto Increment** برای مدیریت شناسه‌های یکتا.
- شامل داده‌های تست (۱۰ رکورد برای هر جدول) جهت بررسی عملکرد.
- دارای کوئری‌های پیشرفته شامل **Joins** و **Stored Procedures**.

## 🛠 تکنولوژی‌های استفاده شده
- **Language:** SQL
- **RDBMS:** MySQL / MariaDB
- **Tools:** MySQL Workbench (for Self-contained file)

## 🗂 ساختار جداول (Database Schema)
دیتابیس شامل جداول اصلی زیر است:
1. **Patients:** اطلاعات هویتی و بیمه بیماران.
2. **Dentists:** اطلاعات پزشکان و تخصص‌ها.
3. **Services:** خدمات ارائه شده در کلینیک (عصب‌کشی، جرم‌گیری و ...).
4. **Appointments:** مدیریت زمان‌بندی و نوبت‌های ویزیت.
5. **MedicalRecords:** تاریخچه درمانی و تشخیص‌های پزشکی.
6. **Insurances:** شرکت‌های طرف قرارداد و درصد پوشش.
7. **Installments/Payments:** مدیریت امور مالی و اقساط بیماران.
8. **Staff:** اطلاعات پرسنل اداری و دسترسی‌ها.

## 🚀 نحوه استفاده
برای راه‌اندازی پروژه در سیستم خود، مراحل زیر را دنبال کنید:

1. فایل `Self-contained SQL File` را دانلود کنید.
2. در **MySQL Workbench** به منوی `Server` و سپس `Data Import` بروید.
3. گزینه `Import from Self-Contained File` را انتخاب کرده و مسیر فایل را بدهید.
4. دکمه `Start Import` را بزنید تا جداول، روابط و داده‌ها ساخته شوند.
